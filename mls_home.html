Overview

MLS (Memory List System) brings the ease of high-level languages to C. By using Integer Handles, it eliminates common memory pitfalls while maintaining blazing performance.

⚡ Handle-Based

Reference data with stable IDs. Resizing memory never invalidates your handles. No more stale pointers or manual remapping.

🔄 Auto-Cleanup

Build complex, nested data structures and free them with a single call. Built-in cycle safety ensures zero leaks.

🛡️ Secure

Integrated bounds checking and Use-After-Free detection. If your code compiles, it is already significantly safer.
