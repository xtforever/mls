◆ ticker_t

typedef struct ticker_s ticker_t

A ticker makes it easy to count-down events until some limit. You ticker_init the ticker to trigger every nticks events. You then notify it that an event has occurred with calls to ticker_tick (or that nticks events have occurred with a call to ticker_ticks), which will return true (and reset the counter) if the countdown hit zero.
