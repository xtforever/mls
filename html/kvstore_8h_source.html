12 * KVSTORE
13 * -------
14 * Index-based KV store implementation. This file implements a KV store comprised
15 * of an array of dicts (see dict.c) The purpose of this KV store is to have easy
16 * access to all keys that belong in the same dict (i.e. are in the same dict-index)
17 *
18 * For example, when Redis is running in cluster mode, we use kvstore to save
19 * all keys that map to the same hash-slot in a separate dict within the kvstore
20 * struct.
21 * This enables us to easily access all keys that map to a specific hash-slot.
22 *
23 * Portions of this file are available under BSD3 terms; see REDISCONTRIBUTIONS for more information.
24 */
