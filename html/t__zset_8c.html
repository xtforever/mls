◆ zrangeGenericCommand()

void zrangeGenericCommand ( zrange_result_handler *  handler,
int  argc_start,
int  store,
zrange_type  rangetype,
zrange_direction  direction 
)

This function handles ZRANGE and ZRANGESTORE, and also the deprecated Z[REV]RANGE[BYSCORE|BYLEX] commands.

The simple ZRANGE and ZRANGESTORE can take _AUTO in rangetype and direction, other command pass explicit value.

The argc_start points to the src key argument, so following syntax is like: <src> <min> <max> [BYSCORE | BYLEX] [REV] [WITHSCORES] [LIMIT offset count]
