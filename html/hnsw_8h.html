hnswNode * hnsw_random_node ( HNSW *  index,
int  slot 
)

Return a random node from the HNSW graph.

This function performs a random walk starting from the entry point, using only level 0 connections for navigation. It uses log^2(N) steps to ensure proper mixing time.
